# ocr-wordsearch-solver

To compile:
```
make all
```
To clean:
```
make clean
```
## Image preprocessing
```
./grid_extract images/'name_of_image' angle
```
To delete all the created images
```
make extra-clean
```

## Solver
```
./solver grid 'word to search'
```

## Neural network
```
./xor
```
